# 제 2강 - 사원수(Quaternioun)과 행렬(Matrix)
## 수업요약
- 사원수의 특징을 이해한다.
- 행렬의 계산법을 익힌다.
- 오일러 회전과 사원수 회전의 장단점을 이해한다.

## 수업결과물
- 회전 가능한 3차원 벡터 클래스 Vector3D를 만든다.
  - x, y, z 축 회전각을 인자로 회전하는 함수를 구현한다.
  - 임의의 축으로 원하는 각도로 회전하는 함수를 구현한다.
